Project Overview

Coffee House is a fully responsive coffee shop website built with HTML, CSS, and JavaScript, focused on delivering a warm and inviting user experience. The site includes a hero section, about page, dynamic menu display, customer testimonials with a slider, image gallery, and a contact form. Interactive navigation, modern UI design, and smooth layout structure demonstrate attention to usability, responsiveness, and front-end best practices.
